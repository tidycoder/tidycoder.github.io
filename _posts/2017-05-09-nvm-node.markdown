###使用nvm管理node版本号
先下载并安装nvm, 需要打开心的shell，才能使用nvm

nvm install v7.7.3

nvm use v7.7.3

然后就可以happy的使用v7.7.3了。

另把.npmrc放到当前目录， 可以直接起作用。 比如说配置淘宝源。
